Sleep Stats

Designer and Developer

The Process

Sleep Stats was one of my first creative websites that I designed and developed. The goal behind this project was to create a website to advertise a product, real or fake, allowing for a lot of creative freedom. This was also one of my first times using CSS3 and beginning to learn animations with CSS

The Result

The end Result was a three page website promoting a fictional sleeping device that would track your sleeping pattern and help you sleep. The creative freedom and discovering new ways to use animation with code thought me a lot about design.
